# CD-RISC 회복탄력성 — sohyunio 인수인계

## 이게 뭔가
CD-RISC(회복탄력성)를 **개념 단위 카드**로 정리한 신 스키마 v1 초안입니다. 4장:
통제감 / 변화수용·적응 / 끈기·회복력 / 안정적 관계·지지.

## 왜 여기 있나 (핸드오프 사유)
- WORKLOG §8 · lanollab README §9 기준, **CD-RISC 회복탄력성은 원래 sohyunio 담당**
  (`paper_rag.py` → `psychology_db.pkl`, 논문 청킹 RAG 방식).
- 이 카드 버전은 "원본 카드가 없다"고 보고 minjub이 이론 근거로 작성했다가,
  담당이 겹친다는 걸 확인하고 **활성 카드 세트에서 분리**한 것입니다.
- 즉 이 폴더의 파일은 **psych_theory 벡터DB에 적재되지 않습니다**
  (로더 `build_psych_cards_db.py`는 상위 폴더의 `cards_*_v1.json`만 읽고, 하위폴더는 제외).

## sohyunio가 결정할 것
1. 회복탄력성을 **논문청킹(기존 방식)** 으로 갈지, **개념카드(이 초안)** 로 갈지, 혼합할지.
2. 이 카드 초안을 참고/입력으로 쓸지, 버리고 처음부터 갈지.

## 주의
- ⚠️ **AI가 이론(Connor & Davidson, 2003 등) 근거로 작성한 초안** — 실제 논문 대조 검증 안 됨. 팀 검수 필요.
- CD-RISC 척도 원 문항은 저작물이라 미포함, 요인 개념만 재구성.
- 스키마·필드 규약은 상위 `../SCHEMA.md` 참조.
