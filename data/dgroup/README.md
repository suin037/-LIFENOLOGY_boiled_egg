# Parallel Me — D그룹 데이터 전처리 산출물

이직·창업·진학 등 인생 선택 시뮬레이션 서비스 "Parallel Me"의 **룰베이스(Layer 1) 조회 테이블**과 **RAG(시나리오 생성 근거) 청크**를 만드는 D그룹 데이터 전처리 결과물입니다.

- 담당 범위: 임금·창업·교육·삶의 질 관련 공공통계 8종 + 웰빙 심리학 논문 3편
- 최종 산출: RAG 청크 **123개**, lookup 테이블 **8개**, 변수 사전 **7개**

---

## 1. 파일 네이밍 규칙

`[용도]_[데이터셋약칭]_[내용]_v버전.확장자`

| 접두사 | 용도 | 배포 여부 |
|---|---|---|
| `lookup_` | Layer 1 룰베이스 조회 테이블 (pandas로 직접 읽음) | ✅ 서비스 배포 |
| `rag_` | ChromaDB 적재용 청크 JSON | ✅ 서비스 배포 |
| `raw2tidy_` | 원본을 풀어낸 중간 산출물 | 보관용 (재집계 시 사용) |
| `dict_` | 변수 사전 (팀 템플릿 형식) | 팀 문서 |

같은 데이터셋의 dict는 **최고 버전 하나만 유효**합니다 (예: dict_elfs_v3 = 세 표 통합본).

## 3. RAG 청크 스키마

```json
{
  "id": "wage_2025_~29_전체근로자",
  "document": "2025년 고용형태별근로실태조사 기준, 29세이하 전체근로자의 월임금총액은 약 269만원...",
  "metadata": {
    "indicator": "경제안정 | 성장가능성 | 삶의질",
    "source": "출처 조사명 (기관, 표 ID)",
    "year": 2025,
    "age_group": "~29 | 30-39 | 25-35 | 19-34 | 전체 등",
    "topic": "검색 필터용 소주제",
    "doc_type": "통계 | 논문개념"
  }
}
```

- 청크 1개 = 통계 문장 1개. 문장 안에 연도·연령·출처가 포함되어 임베딩만으로도 검색됩니다.
- 일부 청크에 `ksic_section`(산업 조인 키), `emp_type`, `major_field` 등 추가 메타데이터가 있습니다.
- 임베딩 모델 권장: `jhgan/ko-sroberta-multitask` (768차원, 한국어 문장 유사도 검증됨).

## 4. 데이터셋 간 조인 키 (팀 표준)

| 키 | 값 | 사용처 |
|---|---|---|
| `ksic_section` | KSIC 10차 대분류 A~S 문자 | elfs 산업별 ↔ bizsurvival ↔ (GOMS·워크넷 권장) |
| `age_group` | "~29", "30-39" 등 문자열 | 전 데이터셋 |
| `year` | 4자리 정수 | 전 데이터셋 |

## 5. ⚠️ 사용 시 주의사항 (함정 목록)

1. **세전 vs 세후**: elfs 임금은 세전 총액, KLIPS p**1642는 세후. 직접 비교·혼용 금지.
2. **생존율 표의 연도는 측정 기준연도**: 2023년의 5년 생존율 = 2018년 창업 코호트. `birth_year` 파생 컬럼 사용.
3. **생존율은 상용근로자 1인 이상 기업 한정**: 고용원 없는 1인 자영업 미포함이라 전체 창업보다 높게 나타남.
4. **사회적 고립도는 높을수록 나쁨**: 점수화 시 방향 반전 필요.
5. **계층이동 지표 2종 혼용 금지**: 사회조사 27.7%(가능성 높다)와 사회통합 66%(노력하면 그렇다)는 문항·척도가 다름. topic 메타데이터로 구분되어 있으며, 시나리오 프롬프트에서 같은 topic 내에서만 인용할 것.
6. **자살률 통계는 의도적으로 청크에서 제외**: 시나리오 생성 서비스 특성 고려. 팀 논의 없이 추가 금지.
7. **age_group 기준이 지표마다 다름**: 19~34(청년보고서), ~29/30-39(사회조사 계열), 25-35(파생) 등. 반드시 메타데이터 확인.
8. **KEDI major_field는 키워드 근사 분류**: '기타' 12.7%. 학교 식별자 없어 학과 단위 비교 불가(집계 전용).
9. **elfs 산업별 표에는 연령 차원 없음**: 산업별 값은 전연령 평균.
10. **지표누리 원출처 중복 주의**: qol 지표의 원출처가 사회조사·사회통합실태조사라 별도 사회조사 데이터셋은 만들지 않음(간접 커버).

## 6. 원자료(raw) 정책 — 레포 미포함

아래 원본 파일은 재배포 제한 가능성이 있어 **레포에 올리지 않습니다** (.gitignore 처리). 필요 시 출처에서 직접 다운로드:

- 사회통합실태조사 원자료 xlsx/SAV → KIPA 통계DB (kipa.re.kr)
- 정신건강실태조사 보고서 PDF → 국립정신건강센터 (mhs.ncmh.go.kr)
- 청년 삶의 질 2025 보고서 PDF → 국무조정실 보도자료 (mods.go.kr)
- KOSIS 집계표 CSV → kosis.kr (표 ID는 2절 표 참조; 공공누리 출처표시 조건이라 팀 판단으로 포함 가능)
- KEDI 취업통계 CSV → kess.kedi.re.kr
